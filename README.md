# React Concepts Demo

This project is a simple React application demonstrating core React concepts such as:

- JSX Rendering  
- Props & State  
- List Rendering  
- Controlled Forms  
- useEffect Lifecycle  

The project contains modular components, each explaining one concept with a working example.
